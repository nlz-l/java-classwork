## 🚀 使用方式

### 运行 .txt 代码片段

```bash
# 1. 复制到 .java 文件
cp 计算器1.txt Calculator.java

# 2. 确保类名与文件名一致

# 3. 编译运行
javac Calculator.java
java Calculator
```

### 查看期末作品

1. 先阅读 `期末作品/说明文档` 了解项目概况
2. 用 IntelliJ IDEA 打开 `期末作品/` 目录
3. 运行主类入口
4. 参考演示视频了解运行效果
