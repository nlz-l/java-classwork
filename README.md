## 使用方式

### 运行 .txt 代码片段

由于课堂练习以 `.txt` 格式保存，运行前需要：

```bash
# 1. 复制内容到对应的 .java 文件
cp 计算器1.txt Calculator.java

# 2. 调整类名与文件名一致

# 3. 编译运行
javac Calculator.java
java Calculator
```

### 查看期末作品

1. 先阅读 `期末作品/说明文档` 了解项目概况
2. 用 IntelliJ IDEA 打开 `期末作品/` 目录
3. 运行主类入口
4. 参考演示视频了解运行效果

## 整理建议

| 现状 | 建议 |
|---|---|
| 代码以 `.txt` 分散存放 | 按主题归类到 `src/<主题名>/`，改为 `.java` |
| 无包结构 | 添加包声明，便于 IDE 管理 |
| 素材散落 | 统一放入 `assets/images/`、`assets/audio/` |
